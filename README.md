# EventPlanner-UI

**EventPlanner** este o aplicație desktop destinată organizării evenimentelor personale, precum zile de naștere și întâlniri. Aplicația permite utilizatorilor să creeze și să gestioneze evenimente cu ușurință.

## Prototip Figma

Accesează prototipul interactiv al aplicației EventPlanner la următorul [link Figma](https://www.figma.com/design/U81a3tue8rkxSE7PhPTrB3/Untitled?node-id=0-1&t=j72O4nu3f0Xpkzqm-1).

## Descrierea aplicației

EventPlanner permite utilizatorilor să:
- Vizualizeze un calendar simplu, cu 7 coloane și 5 rânduri.
- Adauge evenimente folosind butonul „+ Eveniment”.
- Completeze un formular cu câmpurile „Nume eveniment”, „Dată” și „Oră”.
- Vizualizeze evenimentele salvate pe calendar.

## Cum să rulezi prototipul

1. Deschide link-ul Figma pentru a vizualiza și interacționa cu prototipul.
2. Navighează între calendar și formular folosind butonul „+ Eveniment”.
3. Salvează evenimentele și vizualizează-le pe calendar.
